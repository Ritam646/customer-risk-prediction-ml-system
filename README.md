# Customer Risk Prediction ML System

Production-grade machine learning system to predict customer churn and risk scores.

## Tech Stack
- Python, Scikit-learn, XGBoost
- MLflow (experiment tracking)
- FastAPI (model serving)
- GitHub Codespaces (DevContainer-based setup)

## How to Run
1. Open repo in GitHub Codespaces
2. Environment auto-configures
3. Start development
